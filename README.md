# AI CORE Chatbot 🤖

An AI-powered chatbot application built using **React, Node.js, and the Gemini API**.
The application provides a modern chat interface, voice input support, and conversational AI responses.

---

## 🚀 Features

* AI conversational chatbot powered by Gemini API
* Modern chat interface with typing animation
* Voice input using Speech Recognition API
* Dark / Light mode toggle
* Responsive chat UI
* Real-time AI responses from backend API
* Clean and minimal user interface

---

## 🛠 Tech Stack

### Frontend

* React
* Axios
* React Icons
* CSS

### Backend

* Node.js
* Express.js
* Gemini AI API
